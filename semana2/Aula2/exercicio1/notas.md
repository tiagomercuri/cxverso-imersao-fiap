Prática

Criar um projeto de um portal de Feedback de um Banco Digital com base

no protótipo

Informações:

Os campos precisa ter validação e não pode ser vazio
campo nome (mínimo 3 caracteres)
campo email (precisa validar o email)
campo mensagem mínimo de 10 caracteres
ao clicar em enviar deve gerar uma alerta dizendo  "Obrigado,(nome)! Seu feeddback foi enviado com sucesso"
criar a estrutura do projeto usando src, assets(css e js) e o script.ts(com toda a lógica do projeto)
criar o arquivo tsconfig.json com as configurações mínima
o HTML precisa está totalmente semântico
a estilização precisar estar fiel ao protótipo segue palheta de cores

#1a237e;   /* Azul Marinho */
#ffd600;   /* Amarelo */
#d32f2f;   /* Vermelho para mensagens de erro */
#388e3c;   /* Verde para sucesso */
#f5f5f5;   /*cor de fundo*/
